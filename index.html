Project: School Project with Google Gemini Pro API

Overview: This project utilizes the newly released AI, Google Gemini Pro, for word meaning retrieval. It incorporates the Gemini Pro API and employs the Python library pyttsx3 for converting the retrieved meanings into speech.

Prerequisites

  • For GUI App: pip install google-generativeai flet pyttxs3
  • For Use with Microphone: pip install google-generativeai SpeechRecognition pyttxs3

How to Use

  1. Obtain your API key from Google AI Dev.
  2. Enter your API key.
  3. Start using the application.
